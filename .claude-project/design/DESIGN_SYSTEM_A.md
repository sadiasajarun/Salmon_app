# DESIGN_SYSTEM — Variation A: "Editorial Warmth"

Boutique, magazine-like. Serif display headings, warm ivory canvas, airy whitespace. Client-leaning
premium feel; partner screens stay dense but sit on the same warm system.

## Colour tokens
| Token | Value | Use |
|---|---|---|
| primary | `#800020` | brand, primary CTA, active nav |
| primaryHover | `#66001A` | pressed |
| ink | `#222222` | primary text, dark surfaces |
| inkSubtle | `#5B5350` | secondary text |
| canvas | `#FAF6F1` | app background (warm ivory) |
| surface | `#FFFFFF` | cards |
| line | `#EAE0D6` | hairlines |
| success | `#1E7A46` · warning `#B7791F` · error `#B4232B` · neutral `#6B7280` | status only |

## Type
- Display/H1/H2: serif (e.g. "Fraunces"/"Playfair" family). Display 30/1.15, H1 24/1.2, H2 19/1.3.
- Body/UI: humanist sans (e.g. "Inter"). Body 15/1.5, Caption 13, Micro 11.
- Bengali: same sizes, line-height 1.6.

## Shape & elevation
- Radius: cards 24, buttons 26 (pill), inputs 16.
- Shadow: soft, low `0 8px 24px rgba(34,34,34,.06)`.
- Gutter 20, section gap 24.

## Components
- Buttons: pill, filled maroon / outline maroon. Height 50.
- Status pill: tinted bg (10% of status), label + dot.
- Bottom nav: 5 icons, active = maroon, label always visible.
- Cards: large image top (16:9 or 4:3), overlaid status pill.

## Rhythm
Client = 1–2 primary elements per fold, big media. Partner = same tokens, tighter stacks, KPI tiles
2-up.
